Projeto Python

Projeto desenvolvido para prática de lógica de programação utilizando Python.

Tecnologias utilizadas

- Python

Objetivo

Praticar conceitos fundamentais da programação como variáveis, condicionais, loops e entrada de dados.

O que aprendi

- Estruturas condicionais (if/else)
- Estruturas de repetição
- Organização de código em Python
- Raciocínio lógico
